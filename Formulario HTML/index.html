Perguntas e respostas

a) Na tag < input > com atributo type="text", para que serve o atributo "placeholder"?

Na tag < input > com atributo type="text", o atributo placeholder é usado para exibir um texto de exemplo dentro do campo de entrada. Esse texto fornece uma dica ou instrução para os usuários sobre o que eles devem inserir no campo. Ele desaparece quando o usuário começa a digitar no campo.

b) Na tag < input > com atributo type="radio" ou type="checkbox", para que serve o atributo "checked"?

Na tag < input > com atributo type="radio" ou type="checkbox", o atributo checked é usado para indicar que um botão de opção (no caso do type="radio") ou uma caixa de seleção (no caso dtype="checkbox") deve ser marcado por padrão quando a página carregar. Isso significa que elemento já estará selecionado quando o usuário visualizar a página.

c) Na tag < input > para que serve o atributo "value"?

Na tag < input >, o atributo value é usado para especificar o valor inicial do campo de entrada. O valor digitado pelo usuário ou selecionado em um menu suspenso é geralmente enviado como parte do formulário quando o formulário é enviado para o servidor.

d) Que atributo deve ser utilizado para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez?

Para agrupar vários botões de rádio em um formulário de modo que apenas um deles fique marcado de cada vez, você deve usar o atributo name com o mesmo valor para todos os botões de rádio que você deseja agrupar. Isso permite que o navegador identifique que esses botões fazem parte do mesmo grupo e garante que apenas um deles possa ser selecionado ao mesmo tempo.

e) Para que serve a tag < label >? Para que serve o atributo "for" nessa tag?

A tag < label > é usada para rotular elementos de formulário, como campos de entrada ou botões. O atributo for nesta tag associa o rótulo ao elemento de formulário correspondente por meio do atributo id do elemento. Isso melhora a acessibilidade e a usabilidade, permitindo que os usuários cliquem no rótulo para focar no elemento de formulário relacionado.

f) Na tag < option > para que serve o atributo "selected"?

Na tag < option >, o atributo selected é usado para especificar qual opção de um menu suspenso (tag < select >) deve ser selecionada por padrão quando a página carregar. Apenas uma opção pode ser marcada como selected em um menu suspenso.

g) Nas tags de formulário para que serve o atributo "required"?

Nas tags de formulário, o atributo required é usado para indicar que um campo é obrigatório e não pode ser deixado em branco pelo usuário. Se um campo com required estiver vazio quando o formulário for enviado, o navegador exibirá uma mensagem de erro e não permitirá que o formulário seja enviado até que o campo seja preenchido.

h) Para que serve a tag < textarea >?

A tag < textarea > é usada para criar um campo de entrada de texto multilinha em um formulário. Ela permite que os usuários insiram texto longo, como comentários ou descrições.

i) Na tag < form> para que serve o atributo "action"?

j) Na tag < form >, o atributo action especifica o URL para o qual os dados do formulário serão enviados quando o formulário for submetido. Geralmente, isso é o endpoint do servidor que processará os dados do formulário.

k) Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, qual deve ser o valor do atributo "method" na tag < form >?

Para que o valor digitado de uma senha não apareça na barra de endereços do navegador, você deve usar o método "POST" no atributo method da tag

l) Qual valor deve ser informado no atributo "method" na tag < form > em um formulário queserá usado para fazer upload de arquivos??

Para fazer o upload de arquivos em um formulário, o valor do atributo method na tag deve ser definido como "POST". Os uploads de arquivos geralmente requerem o uso do método "POST" para enviar os dados binários dos arquivos.

As tags < fieldset > e < legend > são usadas para criar grupos de elementos de formulário e fornecer uma legenda ou rótulo para esse grupo. Isso melhora a organização e a acessibilidade de formulários mais complexos
